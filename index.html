We started the Cliché Project because we realized that there are no videos that are completely done by girls.
So we wanted to change that and create something that we can be proud of and represents BMX in our opinion. The name "Cliché“
came into our minds because on one hand, there are many stereotypes in women's BMX we want to break with this video project.
